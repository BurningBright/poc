## 启用swagger

在application.properties中添加如下配置：

```
spring.swagger.enabled=true
```

## 启用swagger-ui

在application.properties中添加如下配置：

```
spring.swagger.enabled=true
spring.swagger.ui.enabled=true
```

## 自定义swagger-ui

在application.properties中添加如下配置：

```
spring.swagger.enabled=true
